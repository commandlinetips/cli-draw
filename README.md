# cli-draw

Terminal drawing canvas with mouse support using Python curses.

## Run

```bash
uv run cli-draw
```

If mouse dragging does not report movement in your terminal, try a terminal with mouse reporting enabled.

## Controls

- Left click: draw once
- Left hold + move: draw continuously
- `1` / `2` / `3` / `4`: switch brush character
- `c`: clear canvas
- `q` or `Esc`: quit
